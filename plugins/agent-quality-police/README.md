# agent-quality-police

Strict governance framework for coding agents that blocks testing and typing bypasses.

## Install

```bash
npx agent-quality-police install
```

The installer asks which supported tools should be configured and whether it may manage each global root entrypoint.

Supported tools:

- Claude Code
- Codex
- OpenCode
